# python_pdf_report

Simple demo de como usar python para generar reportes en pdf dinamicos utilizando xhtml2pdf y jinja2

> virtualenv env
> source bin/env/activate
> pip install xhtml2pdf jinja2 pathlib
> python generate_pdf.py test.html
